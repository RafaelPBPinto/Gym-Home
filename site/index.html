Smart Gym @ Home

Gym at Home

Sistema de Ginásio em casa para idosos.

Sem treinador

O nosso sistema permite guiar os utillizadores a fazer exercícios.

Sozinho em casa mas não sozinho

Possibilita interagir com outros utilizadores.

Objetivos

Com o envelhecimento da população mundial, é necessário desenvolver sistemas de saúde capazes de apoiar os idosos na sua vida diária e de promover um envelhecimento ativo e saudável. Assim, é importante a definição de estratégias de envolvimento inteligente para persuadir e motivar os idosos a mudanças de comportamento positivas.


Neste projeto, implementou-se um sistema direcionado a idosos que possibilita a prática de exercício físico em comunidade a partir do conforto das suas casas. Acreditamos que o sistema desenvolvido tem potencial para melhorar a qualidade de vida dos idosos aumentando o seu bem-estar físico e, consequentemente, mental.

  • Promover estilo de vida ativo e saudável
  • Possibilitar a prática de exercício físico em comunidade

Funcionalidades

Interação por voz

O Smart gym at Home é um sistema com suporte para a interação por voz.

Transmissão de vídeo

Há suporte para captação e Transmissão de vídeo.

Interface gráfica para utilizadores

Não somente uma Tela vazia, uma interface com usabilidade para utilizadores.

Base de dados

Há uma base de dados com exercícios/planos de treino e mais algumas informações.
